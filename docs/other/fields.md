# Fields

The following fields are supported in migration / model / scaffold generators:

## Text

Use it as `text`

## String

Use it as `string` or `str`

## Integer

Use it as `integer` or `int`

## Float

Use it as `float`

## Boolean

Use it as `boolean` or `bool`

## DateTime

Use it as `datetime`

## Date

Use it as `date`

## Time

Use it as `time`

## JSON

Use it as `json`
